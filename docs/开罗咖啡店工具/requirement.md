这是一个工具名字叫开罗创意咖啡店攻略

这个工具有两个子页面

# 顾客烦恼
元数据是 `docs/开罗咖啡店工具/顾客烦恼攻略.xlsx`
把他转换成json形式存储，并且多加一个是否为是否已完成
在这个页面中展示这个列表，需要支持包含列：序号，顾客，名称，条件，报酬
这个表格需要包含以下功能
1. 支持用名称进行搜索，支持模糊搜索
2. 在条件内，如果涉及到一下格式的，需要把每个饮品的名称链接到菜谱大全中（点击可以直接跳转到菜谱大全中自动输入搜索对应的菜品名称， 同时保留回退按钮可以回到之前选中的那一列）
3. 每一行后面有一个按钮点击可以切换已完成和未完成的状态

# 菜谱大全
元数据是 `docs/开罗咖啡店工具/菜谱大全.xlsx`
把他转换成json形式存储，并且多加一个是否为是否已完成
在这个页面中展示这个列表，需要支持包含列：序号，饮品（菜谱）, 类别，食材1，食材2，装饰物，价格

这个表格需要包含以下功能
1. 支持用饮品进行搜索，支持模糊搜索
2. 每一行后面有一个按钮点击可以切换已完成和未完成的状态


# 测试方式
需要启动服务，并用playwright来模拟浏览器进行全功能测试
